# restcountries

## Technologies
  * Node.js
  * Express
  * Mocha/Chai
  * Eslint
  * Babel

## Installation
 * Ensure you have node 12.x.x installed.

 * Clone the repo

 * cd into the project
 
 * Install node modules with the command
 
   * yarn install
   
 * Start the server with command
 
   * yarn start
   
 * Check API index with Postman
 
   * http://localhost:3004/api/v1
   
 * Run test with
 
   * yarn test
